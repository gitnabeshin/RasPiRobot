Raspberry Pi Robot
====

## Overview
* Original Robot using Raspberry Pi 3 or ZeroW.
  * 1 Ultrasonic distance sensor
  * 1 SG90 Servo
  * 2 Motor controller(TA7191P) 
  * 6 LED
  * 1 Speaker
  * Direct connector for Raspberry Pi Zero3 or ZeroW
